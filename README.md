# B站评论批量删除

一键批量删除哔哩哔哩（Bilibili）所有评论的油猴脚本。

> **工作原理**：通过 [AICU](https://api.aicu.cc) 第三方 API 按用户 UID 查询全部评论记录（包括视频、动态、专栏），然后调用 B站官方删除接口逐条删除。

## 安装

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展
2. 点击 [安装脚本](https://github.com/Smith-106/bilibili-/raw/main/bilibili-comment-deleter.user.js)，Tampermonkey 会自动弹出安装提示
3. 点击「安装」即可

## 使用方法

1. 打开任意 `bilibili.com` 页面，右下角出现 🗑️ 按钮
2. 点击按钮打开操作面板
3. 点击「📥 获取全部评论」拉取你的所有评论
4. 勾选要删除的评论（或全选）
5. 点击「🔥 删除选中」或「💣 删除全部」

## 功能

| 功能 | 说明 |
|------|------|
| 全量获取 | 通过 AICU API 获取用户在视频、动态、专栏等所有评论 |
| 全选/单选 | 勾选要删的，支持全选 |
| 删除选中/全部 | 两种模式，全部删除有二次确认 |
| 速率控制 | 可调间隔（默认 400ms），触发限频自动冷却 |
| 停止按钮 | 随时中断获取或删除过程 |
| 实时日志 | 显示每条操作结果 |

## 注意事项

- 使用前必须**登录 B 站**
- 脚本通过 `bili_jct` Cookie 鉴权，不会上传任何凭证
- 删除操作**不可撤销**，请确认后再执行
- AICU API 是第三方服务，数据可能有延迟或遗漏
- 如果遇到 AICU 风控，脚本会自动尝试备用接口
